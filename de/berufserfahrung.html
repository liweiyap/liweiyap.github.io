Berufserfahrung

Android-Entwickler

{% include experience-props-lists/props-ad.html %} {% include experience-skills-lists/skills-ad.html %}

Ich entwickle in Java and C++ in unserer nativen Android-App für den Fernzugriff neue Funktionen, u.a. den Dateimanager und die Kontoanmeldung.

Ich verbessere die aktuellen Funktionen, u.a. die Medienprojektion, das SaaS-Lizenzmodell zur Gewinnerhöhung, sowie die Kunden-API in AIDL. Ich behebe auch Fehler in der Nebenläufigkeit und in der UI.

Außerdem arbeite ich zusammen mit einem Grafikdesigner, um die UI zu modernisieren, z.B. habe ich das Drag-and-Drop umgesetzt.

Weiterhin habe ich in C++ ein firmeninternes Programm entwickelt, das während der Kompilierzeit aus der plattformunabhägigen C++ Codebasis Java enums generiert. Das hat eine ganze Quelle Laufzeitfehler eliminiert.

Diese App hat 50+ Millionen Downloads über den Google Play Store. Ich bin mitverantwortlich für Android Code‐Reviews und für die Produktivsetzung von Releases auf den App Stores. Dabei stelle ich rechtzeitige App‐Updates für die neuesten Android‐Versionen bereit, um die jährlichen Android‐Berechtigungsupdates einzuhalten.

Alles in allem ist mir die Entwicklung auf vielen Gerätarten vertraut: Handys, Tablets, Foldables, Android-TVs, Chromebooks und AOSP-Geräten.

Softwareentwickler

{% include experience-props-lists/props-hd.html %} {% include experience-skills-lists/skills-hd.html %}

Ich habe in Java und C++ eine firmeninterne native Android-App zur Fernbedienung eines Audiogeräts entwickelt.

Ich habe mich auch mit der Linux-Entwicklung in C++17 beschäftigt. Dazu habe ich u.a. die UI-Entwicklung mit Qt und die Datenserialisierung mit FlatBuffers gemacht. Außerdem war ich für die Paketverwaltung in Python verantwortlich.

Meine Leistungen sind in Demos präsentiert worden, die dem Start-up Förderungsgeld in Höhe von £715.000 gewonnen haben.

Modellierungs- & Simulationsanalyst

Ich habe in R einen Skript-basierten Workflow zur Simulation von PK/PD Daten für die Auswertung der Wirksamkeit eines Medikaments entwickelt. Diese Daten habe ich mit einer Art statistichem Modell namens „gemischtem Modell“ analysiert. Ich habe solche Modelle mit Monolix geschaffen.
