My CPU

Simulation of a simple stack-based CPU in C++

The CPU contains an operand stack and a register. It processes a given code as a set of 32-bit hexadecimal instructions, which are parsed and disassembled into an opcode and an operand. See the README for an example.

The stack is used to store the results of the elementary operations that are executed according to the opcode and operand. I designed my own stack to make the code easier to debug. The code was compiled using CMake and tested using the Catch2 header-only library.

Predicting Protein Folding Is Difficult

Simulation of the folding of a polypeptide in Matlab

Proteins are large molecules made of building blocks known as amino acids. The prediction of the final folded structure of a protein from just its amino acid sequence is a major unsolved problem in biology. In fact, in computer science, protein folding is considered an NP-hard problem.

In this project, I simulated a simple model of protein folding by applying the Metropolis condition, which probabilistically accepts a trial move of any amino acid residue based on the energy of its resulting interactions with other residues.
