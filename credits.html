Site Credits

Built with Jekyll and hosted on GitHub Pages.

Design based on Hyde theme by Mark Otto. Inspiration also derived from Anjana Vakil, Afonso Raposo, Nathan Randecker, and Vadym Markov.

Other libraries used include:
  • Typewriter JavaScript library from Tameem Safi.
  • Native Carousel CSS and JavaScript libraries from Radoslav Sharapanov.
  • Particles JavaScript library from Vincent Garreau.

The PT Sans, Abril Fatface, and Inconsolata fonts used on this website were taken from Google Fonts.

The icons of the programming languages and technologies were made by Simple Icons and taken from JSDELIVR.

The UK and Germany flag icons were made by Freepik and taken from www.flaticon.com. The external-link icon was taken from the Interface Thick Outline Icons Vectors collection on www.svgrepo.com.

The remaining icons, including social logos, were made by Font Awesome.

The drawing explaining the rules of Conway's Game of Life is from Manuel Küblböck.
